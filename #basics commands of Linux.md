**#basics commands of Linux**





\#**commands**   **:-**  **use of cammands**





* **tty** :- check a terminal type



* **pwd :-** check present working directory



* **hostname :-** check host name / machine name 



* **whoami :-** check login user name 



* **who am I :-** user list show



* **w** **:-** check load average



* **lscpu :-** check hardware related information



* **free :-** check ram related information



* **free -h :-** check ram related information in human language



* **uname :-** check kernel name



* **uname -v :-** check kernel version



* **uname -a :-** check all information about kernel



* **cat/etc/os :-** check software related information



* **clear :-** clear a page



* **ctrl+l :-** shortcut to clear a page



* **date :-** check a current date
* 
**&nbsp;**

* **date -s :-** set the date (ex. date -s 6 jun 2005)



* **cal :-** check current month calender



* **cal particular year :-** check particular year calendar (ex. cal 2005)



* **cal month year :-** check specific month calendar



* **cal -3 :-** check 3 months calendar



* **cal -j :-** check julion days



* **man :-** check command information /provide options



* **q :-** quite the page / exit the page
* 
**&nbsp;**

* **shutdown :-** shoutdown the system after 1 minutes



* **shutdown now :-** shoutdown the system at a time



* **power off  :-** power off the system



* **ctrl+c :-** cancela task



* **init 0 :-** power off



* **switch user :-** switch the user



* **log out :-** log out in user



* **exit :-** exit the page



* **reboot :-** reboot the systeam



* **init 6 :-** shortcut of restart a systeam



* **history :-** check a command history



* **mkdir :-** create a directory

 

* **touch :-** create a file
* 
**&nbsp;**

* **ls -l :-** list the file or directory 
* 
**&nbsp;**

* **ls -a :-**show the files/folder with hiden files/directory 
* 
**&nbsp;**

* **ls -al :-** show all file in detail 



* **mkdir dir1,dir2 / mkdir {dir1,dir2,jgf} / mkdir nikhil{1...10} :-** create a multiple file at a time 



* **mkdir -p a/b/c/d :-** create a parent files



* **cd directory name :-** change a directory
* 
**&nbsp;**

* **cd .. :-** revert directory
* 
**&nbsp;**

* **rm file name :-** delete a empty file



* **rmdir file name :-** delete a empty directory



* **rm -r file name :-** delete a empty/non empty directory/file



* **-r (stand for recursive) :-** 



* **rm -rva file name :-** delete a directory with showing her information



* **rm -r a/b/c/d :-** delete a empty directory / only empty



* **rm -rvf :-** delete a directory forcefully



* **rm \* :-** delete current directory all data 



* **rm -rvf /\* :-** delete main directory all data (do not use)



* **cp source destination :-** copy \& past the file



* **cp -r source destination :-** copy \& past the directory



* **mv source destination :-** move file/directory



* **mv old name new name :-** rename file/directory
